╚═════════════════════════════════════════════════════════════════════════╝

ChatModal (249 lines)
├── Props: { agent: Agent, onClose: () => void }
├── State:
│   ├── inputText: string
│   └── (from useAgentChat hook)
│       ├── messages: Message[]
│       ├── isLoading: boolean
│       └── error: string | null
│
├── Handlers:
│   ├── handleSend() → sendMessage(text)
│   ├── handleKeyDown(e) → Enter/Shift+Enter logic
│   └── handleClearChat() → clearChat()
│
├── Render:
│   ├── AnimatePresence (Framer Motion wrapper)
│   │   ├── Backdrop (onClick → onClose)
│   │   │   └── Modal Container (onClick.stopPropagation)
│   │   │       ├── Header Section
│   │   │       │   ├── Status indicator (●)
│   │   │       │   ├── Agent name + role
│   │   │       │   ├── Clear button (🗑️)
│   │   │       │   └── Close button (×)
│   │   │       │
│   │   │       ├── Messages Area (scrollable)
│   │   │       │   ├── Empty State (when messages.length === 0)
│   │   │       │   ├── MessageBubble[] (mapped from messages)
│   │   │       │   │   └── MessageBubble Component
│   │   │       │   │       ├── Avatar (emoji)
│   │   │       │   │       ├── Message content
│   │   │       │   │       └── Timestamp
│   │   │       │   ├── Loading Indicator (when isLoading)
│   │   │       │   └── messagesEndRef (auto-scroll target)
│   │   │       │
│   │   │       └── Input Area
│   │   │           ├── Textarea (value, onChange, onKeyDown)
│   │   │           ├── Send Button (disabled when loading/empty)
│   │   │           └── Keyboard hint text

MessageBubble (inner component)
├── Props: { message: Message, agentName: string }
├── Render:
│   └── Flex container (justify by role)
│       ├── Avatar (left for agent, right for user)
│       ├── Message bubble
│       │   ├── Content (whitespace-pre-wrap)
│       │   └── Timestamp (role + time)

╔═════════════════════════════════════════════════════════════════════════╗
║                         DATA FLOW                                       ║
╚═════════════════════════════════════════════════════════════════════════╝

USER ACTION: Type "Hola Luna" + Press Enter
    ↓
handleSend()
    ↓
setInputText('') + sendMessage('Hola Luna')
    ↓
useAgentChat.sendMessage()
    ├── setIsLoading(true)
    ├── Add user message to messages[]
    ├── Call sendChatMessage(agentId, { message, sessionKey })
    │       ↓
    │   API: POST /api/agents/:id/chat
    │       { message: "Hola Luna", sessionKey: "abc123" }
    │       ↓
    │   Response: { ok: true, reply: "¡Hola! ¿Cómo estás?", sessionKey: "abc123" }
    │       ↓
    ├── Update sessionKeyRef (for next message)
    ├── Add agent response to messages[]
    └── setIsLoading(false)
        ↓
    Re-render ChatModal
        ↓
    Auto-scroll to messagesEndRef
